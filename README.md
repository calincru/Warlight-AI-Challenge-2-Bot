# AIGames Warlight2 - Aproape Bine team

[C++ Starter repo.](https://github.com/pizzard/warlight-starterbot)

## Build

Make sure you have **gcc-4.9** installed.

    mkdir build
    cd build
    cmake ..
    make

And then just do `make run` to run the bot.
